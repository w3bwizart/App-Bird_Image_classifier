# Bird Image classifier
This app is a converted Jupyter notebook into a Flask app from the course Fast.AI: Practical Deep Learning for coders

## Notebook steps
- The notebook pulls images from the web; birds, forests.
- Create a model
- Train the model
- Validate the model
